# AGENTS.md

## Repo Purpose

Describe the product or system this repository owns.

Keep the statement short enough that a new human or LLM collaborator can orient
quickly.

## Collaboration Guardrails

- write down the current approved focus before widening scope
- keep implementation notes and repo-local docs aligned
- leave handoff notes that another contributor can continue without guesswork
- call out decisions that need explicit review before they spread

## Release Discipline

- keep the release checklist current as work lands
- note what was verified and what still needs review
