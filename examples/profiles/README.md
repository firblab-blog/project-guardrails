# Example Profiles

These directories exist to prove the public extension path without widening the
Rust CLI.

## Available Examples

- [`team-ops`](team-ops)
  shows a fuller custom profile with template overrides, an extra required doc,
  and an additional asset.
- [`human-only`](human-only)
  shows a lightweight collaboration-style example for teams that want a
  human-first review contract.
- [`human-plus-llm`](human-plus-llm)
  shows a lightweight collaboration-style example for teams that want to make
  their human-plus-LLM repo-local contract more explicit.

## Why These Stay As Examples

These examples are intentionally not built-in profiles.
They are small proof points that collaboration style belongs in profile-owned
templates and assets unless repeated real usage justifies widening the public
built-in set.
